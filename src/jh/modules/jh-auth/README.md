# 权限模块

## 调试方式

  在本项目中执行以下代码

  ```
      npm link
  ```

  在需要引入该模块的项目中执行以下代码将依赖删除

  ```
      npm uninstall jh-auth
  ```

  执行以下代码引入本地依赖

  ```
      npm link jh-auth
  ```

  然后正常运行项目 这时修改当前代码 在项目中就会进行热加载

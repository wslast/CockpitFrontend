# 担保-核心模块

## 调试方式

  在本项目中执行以下代码

  ```
      npm link
  ```

  在需要引入该模块的项目中执行以下代码将依赖删除

  ```
      npm uninstall jh-core
  ```

  执行以下代码引入本地依赖

  ```
      npm link jh-core
  ```

  然后正常运行项目 这时修改当前项目代码 在项目中就会进行热加载
